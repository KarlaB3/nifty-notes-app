# Nifty Notes App
Nifty Notes is a note taking app built in React and deployed in Netlify.

**Live App:**
https://nifty-notes.netlify.app/

## About Nifty Notes
Nifty Notes functionality:  
* Lets users create and save a new note with a due date to local storage.
* Lets users edit saved notes with new text, due date and allows users to mark notes as completed. Once completed, they are no longer listed.

## More Information
Nifty Notes development information:  
* Notes are stored as array of objects.
* useContext for global access to notes. 
* useReducer for complex access to notes. 
* useLocalStorage for persistent access to notes.
* useState for forms to make notes.
